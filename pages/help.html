Keyboard Shortcuts

Interface

  • Resize Panels: Ctrl + Shift + 1.
  • Zen Mode: Ctrl + Shift + 2.
  • Resize Editor Width on Zen Mode: Ctrl + Shift + 3.
  • Show time: Ctrl + Shift + X.
  • Show time spend writing: Ctrl + Alt + X.

Markdown Actions

  • Export HTML: Ctrl + Alt + S.
  • Save Markdown: Ctrl + Shift + S.
  • Print: Ctrl + Shift + P.
  • Bold: Ctrl/Cmd + B.
  • Italic: Ctrl + I.
  • Add link: Ctrl + Shift + L.
  • Add image: Ctrl + Shift + K.
  • Copy HTML: Ctrl + Alt + C.
  • Copy RTF: Ctrl + Alt + V.

Text Actions

  • Count Words: Ctrl + Shift + Z.
  • Count Characters: Ctrl + Alt + Z.
  • Uppercase: Ctrl + Shift + U.
  • Lowercase: Ctrl + Shift + I.
  • Capitalize: Ctrl + Shift + O.
  • Hard Line Break: Ctrl + Return.
  • Double Line Break: Shift + Return.
  • Insert "...": Ctrl + ;.
  • Delete all the text: Ctrl/Cmd + Shift + Backspace.

Chrome Integration

TextDown uses and abuse of Chrome APIs. These are some cool things you can do:

  • Offline use.
  • Open any local file, just drag the file to Chrome(go to extension page and check "Allow access to file URLs").
  • Open any web file(go to extension page and check "Allow access to file URLs").
  • Copy a link/image as Markdown from the right-click menu.

Creating List

When creating list you can use tabto create a subitem.

Change Markdown File Extension

You can change the default Markdown file extension. Go to the extensions page on Chrome then go to the options of TextDown.

Change HTML Template

While exporting Markdown as a HTML file TextDown allows you to use a pre defined HTML Template. Go to the extensions page on Chrome then go to the options of TextDown and setup your template.
