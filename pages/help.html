Keyboard Shortcuts

Interface

  • Resize Panels: Ctrl + Shift + 1
  • Zen Mode: Ctrl + Shift + 2
  • Change Editor Width in Zen Mode: Ctrl + Shift + 3
  • Show current time: Ctrl + Shift + X
  • Show time spent writing: Ctrl + Alt + X
  • Go to the top of the page: Ctrl + Shift + 9
  • Stand-alone editor in new window: Ctrl + Shift + Y

Markdown Actions

  • Export HTML: Ctrl + Alt + S
  • Save Markdown: Ctrl + Shift + S
  • Print: Ctrl + Shift + P(uncheck Headers and Footersto print just your document)
  • Bold: Ctrl + B or Cmd + B
  • Italic: Ctrl + I
  • Add link: Ctrl + Shift + L
  • Add image: Ctrl + Shift + K
  • Copy HTML: Ctrl/Cmd + Alt + C
  • Copy RTF: Ctrl/Cmd + Alt + V

Text Actions

  • Count Words: Ctrl + Shift + Z
  • Count Characters: Ctrl + Alt + Z
  • Uppercase: Ctrl + Shift + U
  • Lowercase: Ctrl + Shift + I
  • Capitalize: Ctrl + Shift + O
  • Hard Line Break: Ctrl + Return
  • Double Line Break: Shift + Return
  • Insert "...": Ctrl + ;
  • Insert ". ": Ctrl + .
  • Insert ", ": Ctrl + ,
  • Delete all text: Ctrl/Cmd + Shift + Backspace
  • Change document's name: Ctrl + J
  • Load last document: Ctrl + Alt + R

Some other Features

Open Source

TextDown source code is available on Github.

Smart live Preview

The live preview of TextDown follows what you're typing, so you'll never get lost.

Build-in Calculator

Type {followed by an equation, then press Ctrl + Shift + Mto solve it.

Equation format

You can use functions from the javascript math library, even complex equations such as:

    (2 + 2.4) x Math.sqrt(25)

Chrome Integration

TextDown uses and abuses the Chrome APIs. These are some cool things you can do thanks to that:

  • Work offline.
  • Open any local file. Just drag the file to Chrome, and it'll open for you to edit in TextDown. You'll first need to let TextDown access local files, though, so go to extension page and check "Allow access to file URLs" under TextDown if you'd like to use this.
  • Open any web file. Again, you'll need to enable this from the extension page. Just check "Allow access to file URLs" there.
  • Copy a link/image as Markdown from the right-click menu.

Creating List

When creating list you can use tabto create a subitem.

Github Preview

If you are a developer and wants to write a readme file on TextDown, you can use the Github preview. Go to the extensions page on Chrome, then go to the options of TextDown, and check "Github Preview".

Change Markdown File Extension

You can change the default Markdown file extension. Go to the extensions page on Chrome, then open TextDown's options and set the extension you want.

Change HTML Template

While exporting Markdown as a HTML file, TextDown allows you to use a predefined HTML Template. Go to the extensions page on Chrome, then go to the options of TextDown and setup your template.
